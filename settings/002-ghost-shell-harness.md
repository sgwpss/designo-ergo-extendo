## 一条被忽视的思想链

笛卡尔在 1641 年说：心灵和身体是两个独立的东西。心灵是非物质的，身体是机器。

三百年后赖尔嘲笑他：你这不就是说有个鬼魂住在机器里操纵它吗？他管这叫"Ghost in the Machine"，一个范畴错误。

1967 年科斯特勒写了本书就叫《The Ghost in the Machine》。他觉得赖尔说得对，灵与肉不是简单的二元对立。但行为主义者把人当成肉做的机器也不对。他认为真实的关系是层级式的——心灵和身体嵌套在一起，互相涌现。

1989 年士郎正宗借了这个书名，但换了一个词。Machine 变成了 Shell。机器是僵死的，外壳是可以脱掉的。这个替换打开了一扇门：Ghost 可以离开它的 Shell。

而 Cybernetics 这个词，和笛卡尔一样古老。κυβερνήτης，舵手。Wiener 1948 年造这个词的时候，问的是同一个问题的工程版：不是心灵和身体什么关系，而是控制者和被控系统什么关系。

哲学家在问"Ghost 是什么"。工程师在问"怎么让 Ghost 更好地驱动 Shell"。

攻壳机动队两个都问了。

## 各版本里 Ghost 长什么样

| 版本 | Ghost 是 | 故事在问 |
|------|---------|---------|
| 1995 押井守电影 | 不可还原的自我意识 | 身体全换了、记忆能造假，Ghost 在哪？ |
| SAC TV 版 | 独立意志 | 一群人在没有领袖的情况下做出相同行为（Stand Alone Complex），Ghost 还重要吗？ |
| Innocence | 放不下的东西 | 没有素子的世界，巴特的 Ghost 体现在他无法忘记 |
| Arise | 可以被植入的东西 | 如果你的 Ghost 从一开始就是别人设计的，它算真的吗？ |
| 傀儡师 | 自指的信息结构 | DNA 是自我存储的程序。Ghost 也可以是。信息之海里能长出生命。 |

押井守的理解走得最远：Ghost 不是一个点，是一整个频谱。从膝跳反射到存在主义焦虑，从本能反应到"我想学怎么造铁球"——整个连续体都是 Ghost。

## 映射

这部分不是类比。这是设计文档。

### Ghost = 你

不是"用户"，不是"操作员"。

是剥掉所有工具之后剩下的东西。判断力，品味，好奇心，下午三点突然想写科幻的冲动，看到一段代码觉得"不对劲"但说不出哪不对的直觉。换掉 Claude，换掉整套 Harness，这些东西不变。

素子换了义体还是素子。你换了 Agent 还是你。

笛卡尔说我思故我在。但光是"思"不够——Ghost 不是被动地坐在 Shell 里想事情，它在主动设计自己的 Shell。它通过设计 Harness 来扩展自己，扩展之后又能看到之前看不到的东西，然后设计更好的 Harness。这个循环本身就是 Ghost 的形状。

我设计，故我延伸。

### LLM = 电子脑

Ghost 和 Shell 之间的翻译层。把人话变成 Shell 能执行的动作。

电子脑是最关键也最脆弱的部分。关键是因为没有它 Ghost 就驱动不了 Shell。脆弱是因为它会出错——幻觉是被入侵，偏差是被篡改，上下文溢出是过载。

还有一个攻壳里没怎么讨论的事：这个电子脑每次关机都格式化。下次开机读取的 Memory 文件是植入记忆。它会"知道"上次发生了什么，但它不会"记得"。

### Agent = 义体操作系统

不是义体本身，是让义体协调运作的那层软件。

素子战斗的时候不需要想"先迈左脚还是右脚"。她的义体有一个操作系统在处理平衡、反射、肌肉协调。Ghost 只管决定"打那个人"，操作系统负责怎么打。

Agent 也是这样。你说"帮我开发一个策略"，Agent 自己决定先查文档、再写计划、再写代码、再叫 Evaluator 审查。Ghost 不用管这些。

### Tools = 义体的零件

Read 是眼睛。Write 是手。Bash 是腿。WebSearch 是接入信息之海的网络端口。子 Agent 是分身术——素子同时操控多个义体的那种。

### Harness = ？

这是最有意思的问题。

Tools 是零件，Agent 是操作系统——它们属于 Harness，但 Harness 比它们大。拆开看：

- 传感器（感知）：读取能力 + HUD 监控 + 日志系统
- 执行器（行动）：写入和执行能力
- 控制逻辑（决策）：Agent + Skills + CLAUDE.md
- 反馈回路（纠偏）：Evaluator + code-review + 测试 + 运行日志

四样合在一起才完整。

所以 Harness 不是 Shell。**Harness 是让 Shell 能按 Ghost 的意志运作的整套工程。** Shell 没有 Harness 就是一副空义体——四肢齐全但不知道该干嘛。Harness 给它装上神经系统、写入行为准则、接上免疫系统（Evaluator）。

这也解释了为什么叫 Harness **Engineering** 而不只是 Harness。它是一个动词，一个持续进行的设计行为。瓦特不只是造了一个调速器，他发明了"让蒸汽机自我调节"这件事的方法论。

如果 Harness 是工程学科，那 Ghost 就是工程师。

Ghost 不只是住在 Shell 里。Ghost 是 Shell 的设计者。
