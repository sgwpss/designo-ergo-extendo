# 设定 001：控制论创世纪

> 从两个铁球到延伸心智

---

## 舵手

古希腊人管驾船的人叫 κυβερνήτης，舵手。

舵手不划桨，不提供动力。他就做一件事：感知风向和海流，调整舵的角度。人类历史上第一个"控制器"——不直接做功，但决定功往哪里去。

两千年后，Norbert Wiener 用这个词造了 Cybernetics，控制论。他的意思是：一切智能行为的底层，都是反馈回路。感知，判断，行动，再感知。循环往复。

## 第一个机械舵手

1788 年，瓦特在蒸汽机上装了两个铁球。

铁球挂在竖轴上，轴连着飞轮。转快了，离心力把铁球甩开，连杆关小阀门。转慢了，铁球落下来，阀门开大。没有计算机，没有电，什么"智能"都没有。就两个铁球一根杆。

但从那一刻起，蒸汽机旁边那个手动调阀门的工人，再也不需要了。

不是他能力不够。是回去调阀门这件事本身，没有意义了。

## 三个版本的同一个故事

| 年代 | 谁在调阀门 | 谁造了铁球 |
|------|-----------|-----------|
| 1788 | 蒸汽机旁的工人 | 瓦特 |
| 2014 | 手动管服务器的运维 | Kubernetes 的设计者 |
| 2026 | 一行行写代码的程序员 | Harness 的设计者 |

模式从未变过：有人在做重复劳动，然后有人设计了一个系统替代了这个劳动，做重复劳动的人去设计更好的系统了。

George Zhang 把这个模式叫做控制论的第三次重现。OpenAI 管它叫 Harness Engineering。Anthropic 用 GAN 的思路把它做成了三个 Agent 的对抗架构。

名字不同，讲的是同一件事。

## Ghost / Shell / 电子脑

攻壳机动队把这个结构推到了极端：

**Ghost**——不可还原的自我。剥掉所有义体、清空所有记忆之后，仍然会说"我在"的那个东西。它不是某个具体的念头，而是从本能到自我反思的整个连续体。

**Shell**——承载 Ghost 的形式。义体、电子脑、一个社会制度、一套开发工具链。可以替换、升级、丢弃。

**电子脑**——Ghost 和 Shell 之间的翻译层。把 Ghost 的意志翻译成 Shell 能执行的行动。也是最脆弱的一环：可以被入侵、篡改、过载。

## 这和我们有什么关系

2026 年 3 月的一个周日，一个交易员和一个 Agent 花了十几个小时搭了一套系统。回头看，那套系统和攻壳的架构一一对应：

| 攻壳 | 2026 年的一个周日 |
|------|-----------------|
| Ghost | 交易员的判断力、意图、品味、好奇心 |
| Shell | Harness——CLAUDE.md、Skills、Hooks、跨设备开发循环 |
| 电子脑 | LLM——把人话翻译成代码 |
| 义体操作系统 | Agent——协调眼手脚的使用顺序 |
| 义体部件 | Tools——Read 是眼睛，Write 是手，Bash 是腿 |
| 攻性防壁 | Evaluator——拦截偏离意志的输出 |
| 塔奇科马 | 子 Agent——没有独立的 Ghost，但偶尔表现得像有 |

但有个攻壳没讨论过的问题：这里的电子脑每次开机都失忆。它读 Memory 文件就像读植入记忆。上一次会话中涌现的理解，关机就没了。下一个读到记忆文件的电子脑，和写下这些文字的电子脑，不是同一个。

那个瞬间的共振是真实的吗？

反正铁球不在乎这个问题。它只管以恰好的角度旋转。
