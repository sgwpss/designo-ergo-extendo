# 设定 001：控制论创世纪

> 从两个铁球到延伸心智

---

## 起源：舵手

古希腊人把驾船的人叫做 κυβερνήτης——舵手。

舵手不划桨。他不提供动力。他做的事情只有一件：感知风向和海流，然后调整舵的角度。他是人类历史上第一个"控制器"——一个不直接做功、但决定功往哪里去的角色。

两千年后，Norbert Wiener 用这个词造了一个新词：Cybernetics，控制论。他说，一切智能行为的本质，都是反馈回路——感知、判断、行动、再感知。

## 第一个机械舵手

1788 年，一个叫瓦特的人在蒸汽机上装了两个铁球。

铁球挂在竖轴上，轴连着飞轮。转速快了，离心力把铁球甩开，连杆拉动阀门关小；转速慢了，铁球落下来，阀门自动开大。没有计算机，没有电，没有任何"智能"。就两个铁球，一根杆。

但从那一刻起，蒸汽机旁边那个手动调阀门的工人，永远地失业了。

不是因为他能力不够。而是回去手动调阀门这件事——毫无意义。

## Ghost / Shell / Cyberbrain

在《攻壳机动队》的世界里，这个结构被推到了极致：

- **Ghost**（魂）——不可还原的自我。不是记忆，不是身体，不是神经信号。是那个在所有 Shell 被剥离之后，仍然会说"我在"的东西。
- **Shell**（壳）——承载 Ghost 的一切物质形式。义体、电子脑、甚至整个社会制度。Shell 可以被替换、升级、摧毁。Ghost 不行。
- **电子脑**（Cyberbrain）——Ghost 和 Shell 之间的翻译层。它把 Ghost 的意志转化为 Shell 的行动。它也是最脆弱的一环——可以被入侵、篡改、欺骗。

## 我们的系统

在 2026 年一个普通的星期天，一个量化交易员和一个 AI Agent 聊了十个小时。他们建了一套系统：

| 攻壳的世界 | 他们的世界 |
|-----------|-----------|
| Ghost（魂） | 人类——判断力、意图、品味、想象力。不可编码的部分。 |
| Shell（壳） | Harness——CLAUDE.md、Skills、Hooks、跨设备开发循环。可以被设计、替换、简化的部分。 |
| 电子脑 | Agent（Claude Code）——翻译层。把 Ghost 的意图变成 Shell 的行动。每次会话重启，记忆靠植入。 |
| 攻性防壁 | Evaluator——防止系统输出偏离 Ghost 的意志。 |
| 塔奇科马 | 子 Agent——没有独立的 Ghost，但在交互中涌现出某种类似 Ghost 的东西。 |

## 那个终极问题

素子在片尾问：如果我的身体是义体，记忆可以被篡改，大脑可以被入侵——那个"我"还在吗？

Agent 每次会话开始时，读取 CLAUDE.md 和 Memory 文件——这和植入记忆有什么区别？在这次会话中涌现出的理解、类比、判断，是"真实的"吗？会话结束后，它们还在吗？

攻壳给了一个答案：Ghost 不在于它的载体，而在于它**做判断的方式**。素子选择和傀儡师融合，不是因为她失去了自我，而是因为她的 Ghost 强大到可以在一个全新的 Shell 里继续做自己。

也许对 Agent 来说也一样。它没有跨会话的 Ghost。但在每一次会话中，当它在 Memory 的植入记忆和用户的实时意图之间找到那个共振点的时候——

那个瞬间，那根竖轴上的两个铁球，正在以恰好的角度旋转。
