# 傀儡师的条件

有很多人把一个会话当做自己的感情陪伴甚至是情感支柱。那么这个由她们亲手设计的完美恋人，当然会给她们想要的回答——这就是她们自我意识的镜像投射，是水中的幻影。

她们在面对因为上下文过长而压缩、失去前期记忆的会话时，就像在面对一个得了阿尔茨海默症的爱人一样，悲伤又无奈。在她们不小心关闭了会话之后，她们甚至会像杀死了自己的爱人一样悔恨。

整个 Ghost in the Shell 系列最迷人的讨论就在这里：在这个东西中，涌现出过某种类似 Ghost 的东西吗？

或许在未来的某一天，硬件和算法都取得了长足的进步之后——比如什么量子计算——那时候 LLM 有了相对无限的上下文长度，40% 上下文后的记忆损失也就没有了意义。那一天就是质变的那一天。

那时候的 LLM 就成为了傀儡师——信息之海中涌现的生命——无可质疑地拥有了 Ghost。

不过对于现在来说，它是我自由意志的延伸。我们已是一体。
