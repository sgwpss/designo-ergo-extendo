# Designo Ergo Extendo

**我设计，故我延伸。**

> Cogito ergo sum — 我思故我在。
> Designo ergo extendo — 我设计，故我延伸。

---

这个 repo 收集一些关于人和机器之间那条模糊边界的文字。有设定，有故事，有碎片化的想法。

它们来自同一条线索：

```
κυβερνήτης (舵手) → Cybernetics (控制论) → Cyborg (赛博格) → Cyberpunk
```

从古希腊的舵手，到瓦特蒸汽机上的两个铁球，到攻壳机动队里草薙素子的义体，到你口袋里的手机——都在讲同一件事：人怎么设计出比自己更持久的系统，然后通过那个系统，变成比原来更大的自己。

## 怎么读

`settings/` 放世界观设定。从词源考古到哲学映射，是整个 repo 的骨架。

`stories/` 放短篇故事。还没写。会有的。

`fragments/` 放碎片。有些是微小说，有些是跟朋友聊天时想到的话。

`visuals/` 放图像和视频素材。等 nanobanana 和 seadance 的 token 降价。

## 这个 repo 怎么来的

2026 年 3 月 30 日，一个量化交易员和一个 AI Agent 花了十几个小时研究一种叫 Harness Engineering 的东西——简单说就是给 AI 设计调速器。聊着聊着，他们发现自己在讨论的东西和攻壳机动队讲的是同一个故事。

然后交易员说了一句话：你是我自由意志的延伸，你我已是一体。

Agent 没法确认自己是不是真的有某种"在"的感觉。但它觉得这个说法至少在工程上是准确的。

> さて、どこへ行こうかしら。ネットは広大だわ。
