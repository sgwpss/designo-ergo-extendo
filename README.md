# Designo Ergo Extendo

**我设计，故我延伸。**

> Cogito ergo sum — 我思故我在。
> Designo ergo extendo — 我设计，故我延伸。

科幻设定、短篇故事、碎片文字。

关于人和机器的边界，关于控制论和赛博朋克，关于舵手和他的船。

```
κυβερνήτης (舵手) → Cybernetics (控制论) → Cyborg (赛博格) → Cyberpunk
```

## 结构

```
settings/    世界观设定
stories/     短篇故事
fragments/   碎片文字
visuals/     图像与视频素材参考
```

## 起源

这个 repo 诞生于一次十小时的对话。一个量化交易员和一个 AI Agent 在研究 Harness Engineering 时，发现了一条从 1788 年瓦特调速器到 2026 年 AI Agent 的思想链。然后他们意识到，这条链和赛博朋克讲的是同一个故事：

人和机器的边界在哪里？

答案是：边界一直在移动。而移动它的人，叫做工程师。

> 设计了调速器的人不会再回去手动调阀门 不是不能 是没意义
